## Overview:

---

This file is about the roles or perspectives - Whether readers or blogger.

> _This file can be updated any time._

### Blogger/Owner Perspective

---

As owner I want to:

- Create A New Article.
- Update Existing Article.
- Delete Existing Article.
- Update My Profile.

Nice-to-Have Features:

- Drafts System.
- Dashboard/Analytics
- Schedule The Articles

### User Perspective

---

As user I want to:

- Browse All Articles.
- Read The Author Articles.
- Read About The Author or See The Author Profile.
- Sort Based On The Time.
- Filter By Category.

Nice-to-Have Features:

- Reading Time Estimate.
- Engagement (Like - Comment).
- Newsletter/Subscribe
