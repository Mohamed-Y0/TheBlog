# TheBlog

A modern, full-stack blog platform built with the **MERN Stack** (MongoDB, Express, React, Node.js). This project provides a robust foundation for creating, managing, and sharing articles with a focus on clean architecture and performance.

## 🚀 Version 1.0 Features

### **Backend (Node.js & Express)**

- **Authentication:** Secure JWT-based auth with cookie support and password hashing via Bcrypt.
- **Role-Based Access:** Managed user permissions (Owner, Writer, and User).
- **Article Engine:** Full CRUD functionality with automated SEO-friendly slugs and estimated reading time calculation.
- **Data Integrity:** Schema validation using Mongoose and custom error handling.

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Backend:** Node.js, Express, MongoDB (Mongoose)
- **Security:** JWT, Bcrypt, Validator
- **Logging:** Morgan
